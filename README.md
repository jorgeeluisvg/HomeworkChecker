
# Homework Checker

Just an easy java web scraping project to check if there is any pending homework.


## Authors

- [@jorgeeluisvg](https://www.github.com/jorgeeluisvg)

  
## Requirements 

make sure to have chromedriver inside your windows folder.

```bash 
  Maven
  JDK 8
  chromedriver
```
